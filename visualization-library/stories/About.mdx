    # Implementation Steps

    1. **Add the library**: Include the library via CDN or npm

    **Using npm:**
    ```bash
    npm install sensemaker-viz-components
    ```

    Then import in your JavaScript/TypeScript:
    ```javascript
    import 'sensemaker-viz-components';
    import 'sensemaker-viz-components/dist/sensemaker-viz-components.css';
    ```

    **Using CDN:**
    ```html
    <script type="module" src="https://cdn.jsdelivr.net/npm/sensemaker-viz-components@latest/dist/sensemaker-chart.umd.js"></script>
    ```

    2. **Add Components**: Place the web components in your HTML
    3. **Configure**: Set attributes for data source, view type, and filters
    4. **Style**: Customize appearance using CSS variables

    See the sidebar for detailed component documentation and configuration options.
